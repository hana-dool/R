문자열 함수
================

# paste

입력받은 문자열을 하나로 붙여준다. <br> ***sep*** : 두 문자열 사이를 구분하는 구분자를 써준다. default =
’ ’ ***collapse*** : 여러개의 붙여진 str 을 하나의 str 으로 붙일때 구분자.

``` r
num = 1:10
alphabet = c('a','b','c')
paste(num,alphabet) # paste 를 통해서 두 문자열을 하나로 붙여 주었다. 
```

    ##  [1] "1 a"  "2 b"  "3 c"  "4 a"  "5 b"  "6 c"  "7 a"  "8 b"  "9 c"  "10 a"

``` r
paste(num,alphabet, sep='과') # 두 vector 사이에 구분자를 '과' 로 설정하였다.
```

    ##  [1] "1과a"  "2과b"  "3과c"  "4과a"  "5과b"  "6과c"  "7과a"  "8과b"  "9과c" 
    ## [10] "10과a"

``` r
paste(num,alphabet, sep='') # sep 을 없애서 간격을 붙일 수 있다.
```

    ##  [1] "1a"  "2b"  "3c"  "4a"  "5b"  "6c"  "7a"  "8b"  "9c"  "10a"

``` r
paste(num,alphabet, collapse='',sep='와')  # 스트링이 모두 합쳐진 모습
```

    ## [1] "1와a2와b3와c4와a5와b6와c7와a8와b9와c10와a"

# substr \[특정 위치 문자열 추출\]

주어진 문자열에서 특정 문자열을 추출한다.

``` r
substr('starcraft',1,4) # 1~4 번째의 문자열을 출력한다.
```

    ## [1] "star"

``` r
country = c('korea','japan','china')
substr(country,1,3)
```

    ## [1] "kor" "jap" "chi"

# sub \[글자 바꾸기\]

sub(old, new,x) string x 에서 처음나오는 old 문자를 new 문자로 한번만 바꾼다

``` r
x <- c('your love is not love')
sub('love','first love' ,x) # love -> first love
```

    ## [1] "your first love is not love"

# gsub \[모든글자 바꾸기\]

string x 내의 모든 old 문자를 new 문자로 ‘모두’ 바꾼다.

``` r
x <- c('your love is not love')
gsub('love','first love' ,x) # love -> first love
```

    ## [1] "your first love is not first love"


# union

중복되는 항목 없이, unique 한 항목만 남기기

``` r
str1 <- c("go", "love", "like", "one")
str2 <- c("love", "love", "Python", "one")
union(str1, str2)
```

    ## [1] "go"     "love"   "like"   "one"    "Python"

# intersect

공통되는 항목만 추출

``` r
str1 <- c("go", "love", "like", "one")
str2 <- c("love", "love", "Python", "one")
intersect(str1, str2)
```

    ## [1] "love" "one"

# setdiff

공통되지 않는 항목 추출

``` r
str1 <- c("go", "love", "like", "one")
str2 <- c("love", "love", "Python", "one")
setdiff(str1, str2)
```

    ## [1] "go"   "like"

# setequal\[동질성판단\]

2개 set 에 대한 동일여부 확인

``` r
str1 <- c("go", "love", "like", "one")
str2 <- c("love", "love", "Python", "one")
str3 <- c("love","love","Python","one")
setequal(str1,str2)
```

    ## [1] FALSE

``` r
setequal(str2,str3)
```

    ## [1] TRUE
