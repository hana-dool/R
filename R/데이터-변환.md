# Min-max scailing

\[0,1\] 사이의 값을 가지게 스케일링이 된다.

``` r
# [0-1] 의 값을 가지도록 정규화
k = score_d.f$stat_score 
scale_score2 = (k- min(k))/(max(k)-min(k))
scale_score2
```
